# ZIXI Token Economics

## ZXC — 模擬波動的遊戲代幣

ZXC（子熙幣）是 Zixi Casino 的**模擬資產**，價格完全由內建的 Market Engine 模擬驅動，**非真實區塊鏈代幣**。

### 波動特性

- 市場引擎模擬 20+ 種資產（股票、加密貨幣、大宗商品），價格受內部隨機種子 + 相位週期影響
- 波動率（volatility）依資產類型不同：加密幣 8-12%、科技股 2-5%、商品 1-3%
- 價格走勢可重現、可驗證（基於 FNV-1a 雜湊）
- 每日複利計息（銀行 2% APY）、貸款年利率 4% APR

玩家在模擬市場中交易、避險、套利，所有盈虧均以 ZXC 計價。

## YJC / 傻幣 — 真實上鏈的資產

YJC（佑戩幣 / 傻幣）是部署在 **Solana** 上的真實代幣，具備以下特性：

| 屬性 | 說明 |
|------|------|
| 鏈 | Solana |
| 類型 | SPL Token（可驗證） |
| 合約 | 已部署，鏈上可查 |
| 用途 | VIP 權益、跨平台治理 |
| 供給 | 固定總量，部分由遊戲內兌換鑄造 |

### 為什麼叫「傻幣」？

「傻幣有傻福」—— 這是一個帶有自嘲精神的迷因資產。它不保證暴漲，但作為 Zixi 生態的鏈上權益憑證，持有者享有：

- VIP 房間存取權（需持倉 1 YJC 以上）
- 零手續費交易
- 未來跨平台治理投票權
- 潛在空投與生態激勵

### 真實性

YJC 合約地址已在平台公開，任何玩家均可透過 Solana Explorer 自行查驗持倉與交易紀錄。不同於 ZXC 的模擬屬性，YJC 的每一筆轉帳與持倉都是鏈上不可篡改的真實記錄。
