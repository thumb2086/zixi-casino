## 七、經濟關係

```
1 YJC = 100,000,000 ZXC（固定匯率）
```

```
寶箱定價：每階約 ×5~×10
稀有(500) → 史詩(2K×4) → 傳奇(10K×5) → 神話(100K×10) → 混沌(1M×10) → 深淵(10M×10) →
神諭(100億×1000) → 原始(1,000億×10) → 超越(1兆×10)
```

```
ITEM_BASE_VALUE（非 token 道具估值）= 同階寶箱價格
例如 mythic 寶箱 100K → mythic 道具估值 100K
原始寶箱 1,000億 → 原始道具估值 1,000億
```
